# Slack<---->Discord Bridge 
A bridge we made for Slack to Discord. Bare in mind this is still work in progress.

### Requirements:
* Python3 
* Discord.py
* slack client

### Installation:
`pip install slackclient`
`python3 -m pip install -U discord.py`

### Getting Started:
* Get a Bot key from Slack and Discord
* Fill out the config
* Run the Script
